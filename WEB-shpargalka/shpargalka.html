Семантические элементы

  • <header> - Заголовочная часть документа или секции.
  • <footer> - Нижняя часть документа или секции.
  • <article> - Самостоятельная статья или пост.
  • <section> - Секция документа.
  • <nav> - Навигационная панель.
  • <aside> - Боковая панель или вставка.
  • <figure> - Контейнер для иллюстраций.
  • <figcaption> - Подпись к иллюстрации.

Новые атрибуты и глобальные атрибуты

  • data-* - Хранение пользовательских данных.
  • hidden - Скрывает элемент.
  • contenteditable - Позволяет редактировать содержимое элемента.
  • spellcheck - Включает проверку орфографии.

API

  • Geolocation API - Определение местоположения пользователя.
  • Drag and Drop API - Перетаскивание элементов.
  • Canvas API - Рисование на холсте.
  • Web Storage API - Хранение данных на стороне клиента.
